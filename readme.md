#####Description

It's a simple app with mvc pattern:
- Controllers located in App\Controllers,
- Models located in App\Models,
- All templates, styles, scripts in \templates,
- Settings for connect to db in App\db_config.php,
- Db dump - \task.sql,
- Sql task - \sql_task.md.

#####Server Requirements of the App
- PHP 7.*,
- Apache 2.*,
- Database MySQL 5 or higher.
